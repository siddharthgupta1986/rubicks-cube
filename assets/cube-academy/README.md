# Cube Academy assets

Local bitmap artwork for the Cube Academy command deck and Journey surfaces.

## Files

- `command-deck-backdrop.png` is a wide 16:9 scene with quiet space on the left for the home-screen copy and the cube positioned on the right.
- `journey-chapter-atlas.png` is a six-panel atlas for Orientation, White Cross, First Layer, Middle Layer, Last Layer, and Mastery Trials. Each panel is arranged in reading order, left to right and then top to bottom.
- `cube-academy-emblem.png` is a square cube mark for badges, chapter headers, and compact progression surfaces.

These files are intentionally unreferenced until the Cube Academy shell is implemented. They have no external requests or runtime dependencies and can be loaded directly from disk.

Artwork generated with the built-in Codex image generation capability for issue #147.
