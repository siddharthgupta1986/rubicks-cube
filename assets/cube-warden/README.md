# Cube Warden artwork

These local raster assets were generated for Cube Warden: The Last Route with
OpenAI's built-in image generation tool on 2026-07-23. They are original
project artwork and contain no third-party logos, text, characters, or source
images.

## Assets

- `title-backdrop.jpg`: Aya's route toward the Dawn Vault.
- `route-map.jpg`: the six-sector Broken Archive campaign map.
- `aya-token.jpg`: full-body character artwork for Aya.
- `void-wraiths.jpg`: abstract shard-like Void Wraith pursuit artwork.
- `encounter-chamber.jpg`: empty seal chamber for the interactive cube.

The production files are resized JPEG derivatives of the generated PNG sources
to keep the dependency-free static app lightweight. Interface text and required
state labels remain semantic HTML rather than being embedded in artwork.

## Prompt direction

The shared direction was a painterly independent-game adventure with angular
archive architecture, charcoal and mineral teal foundations, warm gold route
light, restrained red warnings, and physical cube geometry. Prompts explicitly
excluded weapons, gore, copyrighted characters, franchise resemblance, logos,
watermarks, and embedded text.
