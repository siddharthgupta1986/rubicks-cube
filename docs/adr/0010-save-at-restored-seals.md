# Save campaign position at restored seals

Restoring a cube seal creates the campaign's durable checkpoint, saving completed seals, their world changes, performance results, and the associated safe spawn point. Refresh, reopening, and capture return Aya to that checkpoint rather than an exact mid-corridor position. This prevents restoration inside active pursuit or inconsistent door geometry while keeping recovery predictable; settings and optional discoveries may still persist immediately.
